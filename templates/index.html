OCR Micro-Service


Usage:

    1. Call OCR micro-service with url: "http://10.224.18.222:8005/api/".

    2. Input: the path of the origial image(s), and uploaded in form style.

    3. Output:
         For one image: output is a .txt file, including the text information of the uploaded image.
         For multiple images: output is a zip file, including equal number of files to images.

    4. E.g.
        For one image: curl -F "file=@path_to_image" -o name_of_output_file http://10.244.18.222:8005/api/
        For two images: curl -F "file1=@path_to_image1" -F "file2=@path_to_image2" -o name_of_output_file http://10.244.18.222:8005/api/

Note:

    The image(s) must bigger than 600*600 pixels.
