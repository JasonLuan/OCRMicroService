OCR Micro-Service


Goal:

    Extracting text from image(s). By calling the OCR micro-service, user upload an image or multiple images, the service will return related text file(s).

Usage:

    1. Call OCR micro-service with url: "http://10.2424.18.222:8000/api".

    2. Input: the path of the origial image(s), and uploaded in form style.

    3. Output:
         For one image: output is a .txt file, including the text information of the uploaded image.
         For multiple images: output is a zip file, including equal number of files to images.

    4. E.g.
        For one image: curl -F "file=@path_to_image" -o name_of_output_file http://10.244.18.222:8000/api
        For two images: curl -F "file1=@path_to_image1" -F "file2=@path_to_image2" -o name_of_output_file http://10.244.18.222:8000/api
